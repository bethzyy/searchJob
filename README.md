# 职位查找器 v1.0

Windows 11 专用版本 - 根据关键字自动抓取招聘网站职位链接

## 快速开始

### 方法1: 双击启动(推荐)
直接双击 `start.bat` 即可启动程序

### 方法2: 命令行启动
```bash
python gui.py
```

## 功能说明

界面包含4个元素:

1. **网站URL** - 输入招聘网站地址(如 https://www.zhaopin.com/)
2. **关键字** - 搜索关键字,多个用空格隔开(默认: AI 人工智能)
3. **最大职位数** - 限制搜索数量,留空表示不限制
4. **输出文件** - 结果保存的TXT文件路径

## 使用步骤

1. 填写网站URL
2. 填写关键字(如: AI 算法 机器学习)
3. 可选:填写最大职位数
4. 填写输出文件名
5. 点击"开始查找"

## 常见问题

### Q: ChromeDriver Win32错误?
A: 程序已优化,会自动尝试多种方法。如果还有问题:
1. 运行 `install_driver.bat`
2. 或手动下载: https://googlechromelabs.github.io/chrome-for-testing/

### Q: 找不到职位?
A: 尝试:
- 关闭"无头模式"查看浏览器运行情况
- 使用更通用的关键字
- 检查网站是否需要登录

### Q: 程序卡住?
A: 按Ctrl+C停止,或关闭命令行窗口

## 技术栈

- Selenium 4.15+ - 网页自动化
- Chrome WebDriver - 浏览器驱动
- tkinter - 图形界面

## 注意事项

- 首次运行会自动下载ChromeDriver
- 建议开启无头模式提高速度
- 请遵守目标网站使用条款
- 避免频繁请求
