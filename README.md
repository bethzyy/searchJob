# 职位搜索器 v1.0

Windows 专用版本 - 根据关键字自动抓取招聘网站职位信息

## 快速开始

### 方法1: 双击启动(推荐)
直接双击 `start.bat` 即可启动程序

### 方法2: 命令行启动
```bash
python gui.py
```

## 功能特点

- **智能搜索**: 根据关键字自动搜索匹配职位
- **关键字过滤**: 支持多个关键字(空格分隔),匹配任一即可
- **排除功能**: 可排除包含特定关键字的职位(如:校招、实习)
- **数量限制**: 可设置最大搜索数量,避免过度抓取
- **CSV导出**: 结果以CSV格式保存,方便Excel打开
- **实时进度**: 显示搜索进度和已找到职位数量
- **日志记录**: 详细记录搜索过程,便于调试
- **现代界面**: 采用微软Fluent Design风格清爽配色

## 界面说明

界面包含以下功能组件:

1. **网站URL** - 输入招聘网站地址
   - 默认: https://www.zhaopin.com/sou/jl530/kw010G0I8/p1

2. **关键字** - 搜索关键字,多个用空格隔开
   - 默认: AI 人工智能
   - 说明: 职位标题包含任一关键字即匹配

3. **排除关键字** - 排除不需要的职位
   - 默认: 校招 实习生
   - 说明: 职位包含这些关键字将跳过,留空则不过滤

4. **最大职位数** - 限制搜索数量
   - 默认: 30
   - 说明: 留空表示搜索所有职位

5. **输出文件** - 结果保存路径
   - 默认: jobs_result.csv
   - 支持: 浏览按钮选择保存位置

6. **搜索进度** - 实时显示进度
   - 进度条显示完成百分比
   - 显示已找到职位数量

7. **运行日志** - 详细记录搜索过程
   - INFO: 黑色 - 一般信息
   - SUCCESS: 绿色 - 成功信息
   - WARNING: 橙色 - 警告信息
   - ERROR: 红色 - 错误信息

8. **操作按钮**
   - 开始查找: 启动搜索
   - 停止: 中断搜索
   - 清空日志: 清除日志显示

## 使用步骤

1. 填写或修改网站URL
2. 填写搜索关键字(如: AI 算法 机器学习)
3. 可选: 填写排除关键字(如: 校招 实习)
4. 可选: 设置最大职位数
5. 确认或修改输出文件路径
6. 点击"开始查找"
7. 等待搜索完成,查看CSV文件

## 常见问题

### Q: ChromeDriver Win32错误?
A: 程序会自动处理ChromeDriver问题。如果仍有错误:
1. 运行 `install_driver.bat`
2. 或手动下载: https://googlechromelabs.github.io/chrome-for-testing/

### Q: 找不到职位或数量很少?
A: 可能原因和解决方法:
- 关键字太具体,尝试使用更通用的关键字
- 网站需要登录,先手动登录浏览器
- 网站结构变化,需要更新选择器
- 网络连接问题,检查网络设置

### Q: 程序卡住不动?
A: 解决方法:
- 点击"停止"按钮中断搜索
- 检查网络连接是否正常
- 关闭程序重新启动

### Q: CSV文件乱码?
A: 解决方法:
- 使用Excel打开时选择"UTF-8"编码
- 或用记事本/VS Code查看

## 系统要求

- **操作系统**: Windows 10/11 (64位)
- **Python**: 3.8 或更高版本
- **浏览器**: Chrome 浏览器
- **内存**: 建议 4GB 以上
- **网络**: 稳定的网络连接

```csv
职位标题,职位链接,公司名称,工作地点,薪资,发布时间

人工智能研究员,https://www.zhaopin.com/jobs/789012,某某AI公司,上海,25-40K,2024-12-24
```

## 技术栈

- **Selenium 4.15+**: 网页自动化框架
- **Chrome WebDriver**: 浏览器驱动
- **tkinter**: Python图形界面库
- **CSV**: 数据存储格式

## 注意事项

- **合法使用**: 请遵守目标网站的使用条款和robots.txt
- **频率控制**: 避免频繁请求,以免对网站造成压力
- **数据用途**: 抓取的数据仅供个人学习参考
- **无头模式**: 程序固定使用无头模式,提高运行效率
- **自动去重**: 自动去除重复的职位链接
